# Punos

Arma unica / mecanica especial. Sirve para agarrar enemigos. Puede mover enemigos o moverte a ti. Conecta con el cambio de mundo. Velocidad media-rapida.

**Habilidad especial:** si agarras a un enemigo con poca vida o muy debil y lo matas con X cargado, cambias de mundo. Si no, solo generas medio meter.

## Terrestre

| Input | Descripcion |
|---|---|
| X X X | Jab, golpe fuerte, uppercut. |
| X X espera X X | Jab, golpe fuerte, dos golpes fuertes. |
| X cargado | Agarre que baja vida a un enemigo. |
| X cargado sweet spot | Mas dano. Puedes agarrar a dos enemigos si estan cerca. |
| Y cargado | Launcher. Area pequena. |
| Y cargado sweet spot | Shoryuken doble. |

## Aereo

| Input | Descripcion |
|---|---|
| X X X | Izquierda, derecha, golpe con ambos hacia adelante. |
| X espera X X | Izquierda, derecha uppercut, izquierda hacia abajo. |
| X cargado | El agarre igual que en el piso. |
| X cargado sweet spot | Igual que el piso. Mas dano. Puedes agarrar a dos enemigos si estan cerca. |
| Y cargado | Te acercas a un enemigo como que se estiran los guantes. Te acerca o te trae dependiendo del tamano. |
| Y cargado sweet spot | Te los trae aunque sean grandes. |
