# Dagas

Arma de movilidad. Da mas opciones de reposicionamiento. Sirve para perseguir enemigos. Velocidad rapida.

**Habilidad especial:** si no hay nadie cerca, X lanza una daga al enemigo mas cercano. Si presionas X de nuevo, te teletransportas a ese enemigo. Consume meter.

## Terrestre

| Input | Descripcion |
|---|---|
| X X X X X | Corte con una daga, corte con la otra, estocada doble, estocada doble, swing en X. |
| X X espera X | Estocada doble, estocada doble, lluvia de estocadas. |
| X cargado | Lanza dagas omnidireccionalmente. A todos los enemigos que pega, los teletransporta alrededor del usuario. |
| X cargado sweet spot | Lanza el doble de dagas. |
| Y cargado | Launcher. Area pequena. |
| Y cargado sweet spot | Subes con todos los enemigos cercanos. |

## Aereo

| Input | Descripcion |
|---|---|
| X X X | 2 cortes horizontales, 1 corte hacia abajo. |
| X espera X | Empuja hacia abajo a los enemigos con mayor AOE, 1 vuelta, 1 corte en X. |
| X cargado | Lo mismo que en el piso, pero en el aire. |
| X cargado sweet spot | Lo mismo que en el piso. Lanza el doble de dagas. |
| Y cargado | Subes un poco dando vueltas con todos los enemigos alrededor tuyo. |
| Y cargado sweet spot | Subes mas. Aumenta el rango. |
